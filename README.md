<h3 align="center">Context API </h3>

In this project, I built a Todo application using React and the Context API for state management. I created three main contexts: TodoContext, FilterContext, and ThemeContext. In TodoContext, I manage the list of todos, each with an id, text, and completed status, and provide actions to add, toggle, delete, edit, and clear completed todos. The TodoList component displays todos based on the current visibility filter from FilterContext, which lets me switch between “all,” “active,” and “completed” todos using filter buttons. For ThemeContext, I implemented a light and dark theme switcher that updates the app’s appearance, including the main container and key components, with a toggle button to switch themes. I also added persistence by saving the todos and theme to localStorage, so the state is restored when the app reloads. The project demonstrates managing complex state with Context, providing reusable actions, conditional rendering based on filters, theme switching, and state persistence, while ensuring clean and maintainable code structure.
